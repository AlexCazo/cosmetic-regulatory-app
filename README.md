Cette application a pour objectif d’automatiser le contrôle des principales restrictions réglementaires applicables à une formule cosmétique, afin d’assister le responsable réglementaire dans sa prise de décision.

Il s’agit d’une version de démonstration à périmètre volontairement simplifié, illustrant la traduction de contraintes réglementaires en logique décisionnelle structurée et extensible.
